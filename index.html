CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Three ways to style a webpage: CSS inline, internal CSS style sheet, and external CSS style sheets
  • CSS rule: Selector defines the element or attributes to which the rules, or declarations, will apply: Declarations contain two important components; the CSS property we want to apply and the value of the property
  • 'link' element creates the connection to the external file
  • The period preceding 'card' designates this as a class selector. Any HTML element assigned to the class, 'card', will gain these property assignments from the CSS

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout main: switch to the main branch
  • git checkout -b branch name: creates a new branch and switches to it
  • git clone 'paste copied SHH value': clones GitHub repository onto our computer
  • git pull origin main: sync local branch with remote base branch
  • git push origin 'branch name': push our changes to the remote branch

JavaScript

  • Enter your JavaScript notes here
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • JavaScript is a powerful programming language that can add interactivity to a website
  • To declare a variable, we use the "var" keyword
  • When using the "if" statement, code will only execute if the statment meets a condition, or is true
  • Array: group of related data
  • Index: the location of the data item in the array. Arrays are zero-indexed and sequential
  • A "for" loop uses the predicatble pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • A funciton is simply a set of instructions that tells the computer how to perfrom a certain task
  • Two requirements for a JavaScript function: 1.Define the function 2.Call the function
  • Math.floor() and Math.random() are both properties of JavaScript built-in "Math" object that can be used together to generate a random number
